Позиционирование

CSS-позиционирование описывает, как любой из элементов может быть размещен независимо от порядка документа (т.е. извлечен из «потока»)

Свойство position определяет, какой из алгоритмов позиционирования используется для вычисления положения блока. Свойство не наследуется.

Значения свойсвта position:

  • static (по умолчанию) - блок располагается в соответствии с нормальным потоком. Свойства top, right, bottom и left не применяются.
  • relative - блок смещается относительно его нормального положения и не влияет на положение любых следующих блоков. Такое смещение может привести к перекрытию других блоков. Положение блока задается с помощью свойств top, right, bottom и left.
  • absolute - расположение элемента относительно его ближайшего предка с position: relative или относительно документа. Положение блока задается с помощью свойств top, right, bottom и left.
  • fixed (подвид абсолютного позиционирования) - расположение элемента относительно видимой части браузера. Когда страницу прокручивают, фиксированный элемент остаётся на своём месте и не прокручивается вместе со страницей. Положение блока задается с помощью свойств top, right, bottom и left.

Примеры

position: relative


Нормальное расположение элемента в потоке
.relative {
    position: relative;
    top: 40px;
    left: 40px;
}

position: absolute


Нормальное расположение элемента в потоке
.absolute {
    position: absolute;
    top: 100px;
    right: 100px;
}

position: absolute in relative


Нормальное расположение элемента в потоке
.relative-for-absolute {
    position: relative;
}

.absolute-in-relative {
    position: absolute;
    top: 20px;
    bottom: 20px;
    left: 20px;
    right: 20px;
}

position: fixed


Нормальное расположение элемента в потоке
.fixed {
    position: fixed;
    top: 30%;
    left: 30%;
}
