Псевдоэлементы

Псевдоэлементы - это ключевое слово, добавляемое к селектору, которое позволяет стилизовать определённую часть выбранного элемента

Синтаксис:

селектор::псевдо-элемент {
    свойство: значение;
}

Первая буква элемента

.first-letter::first-letter { /* первая буква элемента '.first-letter' */
    color: red;
}

Первая строка элемента

.first-line::first-line { /* первая строка элемента '.first-line' */
    color: orangered;
    text-transform: uppercase;
}

Selection - позволяет применить стили к части документа, который был выделен пользователем (например, с помощью мыши)

  • color, background-color 
    .selection .color::selection {
    color: yellow;
}
.selection .background-color::selection {
    background-color: orangered;
}
  • text-shadow 
    .text-shadow::selection {
    text-shadow: 5px 5px 10px black;
}

Before / After

before - псевдоэлемент, который является первым потомком выбранного элемента (по умолчанию строчный)

.before::before {
    content: "♥";
    color: red;
    margin-right: 5px;
}

after - псевдоэлемент, который является последним потомком выбранного элемента (по умолчанию строчный)

.after::after {
    content: "\2B50";
    margin-left: 5px;
}

Псевдоклассы

Псевдокласс CSS - это ключевое слово, добавленное к селектору, которое определяет его особое состояние, например, нажатая ссылка

Синтаксис:

селектор:псевдо-класс {
    свойство: значение;
}

Для ссылок:

  • селектор:visited {стили} - ссылка, по которой был осуществлен переход
  • селектор:link {стили} - ссылка, которая ещё не была посещена

Для ссылок и других элементов:

  • селектор:active {стили} - элемент, активировани нажатием мыши, tab и пр
  • селектор:hover {стили} - срабатывает, когда пользователь наводит на элемент мышью!

Для элементов формы:

  • селектор:focus {стили} - применяется, когда элемент (такой как input формы) получает фокус
  • селектор:checked {стили} - input с типами radio и checkbox, option, которые выбраны
  • селектор:default {стили} - элемент формы, установленный по умолчанию для группы связаных элементов (radio или checkbox)
  • селектор:disabled {стили} - элемент, с установленным атрибутом disabled
  • селектор:enabled {стили} - элемент, без атрибута disabled
  • селектор:optional {стили} - элементы, для которых не был установлен атрибут required
  • селектор:required {стили} - элементы, для которых был установлен атрибут required
  • селектор:valid {стили} - поля, которые прошли html валидацию
  • селектор:invalid {стили} - поля, которые не прошли html валидацию
  • селектор:in-range {стили} - элементы, значение которых соответствует указанному для них диапазону ограничений
  • селектор:out-of-range {стили} - элементы, значение которых не соответствует указанному для них диапазону ограничений

Дочерние элементы:

  • селектор :first-child - первый дочерний элемент
  • селектор :last-child - последний дочерний элемент
  • селектор :nth-child(число) - например, :nth-child(2) - дочерний элемент под номером 2
  • селектор :nth-child(odd) - все нечетные дочерние элементы
  • селектор :nth-child(even) - все четные дочерние элементы

Стилизация элементов, к которым был осуществлен переход по гиперссылке:

селектор:target - позволяет связать гиперссылку 'a' с любым элементом на странице
